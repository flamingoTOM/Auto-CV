# Auto-CV 视频文稿

> 平台：YouTube / 抖音 / 小红书
> 时长：约 60 秒
> 形式：口播 + 屏幕录制

---

## 分镜脚本

---

### 【0:00 - 0:06】开场钩子

**🎙️ 口播：**
> 写简历这件事，我帮你用 AI 彻底搞定了。

**📹 画面：**
- 快速剪辑：一份空白 Word 文档 → 一份排版精美的 PDF 简历
- 右下角出现标题字幕：**「Auto-CV | AI 简历生成器」**

---

### 【0:06 - 0:16】痛点 + 项目介绍

**🎙️ 口播：**
> 每次投简历都要手动排版、调格式，花好几个小时。
> 我做了一个工具，叫 Auto-CV，用自然语言描述你自己，AI 自动帮你写成 LaTeX 格式，一键编译成 PDF。

**📹 画面：**
- 屏幕录制：打开项目的 GitHub 页面，停留 1-2 秒
- 切到 VS Code，展示项目目录结构（`resume.tex`、`.claude/skills/Auto-CV/`）

---

### 【0:16 - 0:30】核心功能演示①：导入已有简历

**🎙️ 口播：**
> 你已经有一份简历了？没问题，直接把 PDF 丢进去，AI 自动解析、提取内容，转换成模板格式。

**📹 画面：**
- 屏幕录制：在 Claude Code 终端输入 `/Auto-CV 参考 武汉理工大学-刘晓煜.pdf`
- 展示 AI 开始分步骤读取内容（逐条显示解析过程）
- `resume.tex` 文件被自动更新（编辑器中文件高亮刷新）

---

### 【0:30 - 0:46】核心功能演示②：对话式交互生成

**🎙️ 口播：**
> 如果你从零开始，它会引导你走完 7 个步骤——教育背景、实习经历、项目经历……
> 你用大白话告诉它，它帮你写成规范的简历语言，实时更新 tex 文件。

**📹 画面：**
- 屏幕录制：展示 Claude Code 对话框，AI 提示「请填写第 1 步：个人信息」
- 用户输入自然语言（如「武汉理工大学，测控技术与仪器，本科」）
- 切到 VS Code 编辑器，展示 `resume.tex` 中对应区块被自动填入内容

---

### 【0:46 - 0:55】效果展示：PDF 输出

**🎙️ 口播：**
> 全部填完之后，一行命令编译，直接出 PDF。你看，这就是最终效果。

**📹 画面：**
- 屏幕录制：运行编译命令（终端闪过编译日志）
- 打开 `resume.pdf`，全屏展示成品简历（排版清晰、中文字体美观）
- 缓慢上下滚动简历页面，突出排版质量
